# KthSocial

A simple front end for a KTH Social database project

To run:
- Install postgresql
- Execute the sql/create.psql and sql/inserts.psql files
- Install PostgREST
- Start PostgREST with the command: postgrest config.conf
- Install npm and vite
- Start this app with the command: npm run dev
- View the app at http://localhost:5173
